Información sobre la descarga

En esta sección encontrarás detalles sobre el archivo disponible para descarga.

¿Cómo descargar el archivo?

Simplemente haz clic en el botón de descarga que aparece en la página principal. Esto descargará automáticamente un archivo en formato .csv a tu dispositivo.

¿Qué contiene el archivo?

El archivo contiene una base de datos con nombres y su respectivo hash, generados con algoritmos como MD5, SHA-1, SHA-256, entre otros. Este contenido puede ser utilizado para aprender y realizar pruebas relacionadas con funciones hash.

Estructura del archivo

La base de datos está organizada en formato de texto delimitado por comas (.csv), lo que la hace fácil de manipular en herramientas como Excel, Google Sheets o con lenguajes de programación como Python.

  • Columna 1: Nombre o texto original.
  • Columna 2: Hash generado (según el algoritmo).
