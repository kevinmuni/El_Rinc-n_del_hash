Información sobre la Base de Datos

En esta sección encontrarás detalles relacionados con nuestra base de datos de hashes.

¿Qué contiene la Base de Datos?

Nuestra base de datos incluye nombres y sus correspondientes hashes generados con algoritmos como MD5, SHA-512, SHA-1 y SHA-256.

Estructura de la Base de Datos

La información está organizada en formato tabular, ideal para ser utilizada en pruebas o aprendizaje sobre funciones hash.

  • Columna 1: Nombre o texto original.
  • Columna 2: Hash generado (según el algoritmo).
