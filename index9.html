Información sobre la Base de Datos

Esta Base de Datos llamado nombres_hash_60 contiene nombres con varios tipos de hash


¿Qué contiene la Base de Datos?

La Base de datos contiene Nombres y sus correspondientes hash's generados con algoritmos como MD5, SHA-512, SHA-1 y SHA-256.

Estructura de la Base de Datos

La información está organizada en formato tabular, ideal para ser utilizada en pruebas o aprendizaje sobre funciones hash.


  1. Columna 1
    • Contiene Nombres en texto original
  2. Columna 2
    • Contiene Los Nombres en MD5
  3. Columna 3
    • Contiene los nombres en SHA-1
  4. Columna 4
    • Contiene los nombres en SHA-256
  5. Columna 5
    • Contiene los nombres en SHA-512
